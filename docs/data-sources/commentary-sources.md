# Commentary Sources

## Matthew Henry's Commentary

### Source Information
**Author**: Matthew Henry (1662-1714)
**Title**: "An Exposition of the Old and New Testaments"
**Copyright**: Public Domain
**Availability**: Christian Classics Ethereal Library (CCEL)

### Characteristics
- **Style**: Devotional and practical
- **Approach**: Verse-by-verse exposition
- **Length**: Comprehensive (often 500-2000 words per verse)
- **Focus**: Practical application and spiritual insight
- **Theological Perspective**: Reformed/Puritan tradition

### Content Structure
- Verse-by-verse commentary
- Practical applications
- Cross-references to other passages
- Devotional insights
- Historical context

### Data Source URLs
- CCEL: `https://ccel.org/ccel/henry/mhc`
- Bible Study Tools: `https://biblestudytools.com/commentaries/matthew-henry-complete/`
- Archive.org: Public domain digitized versions

## John Gill's Exposition

### Source Information
**Author**: John Gill (1697-1771)
**Title**: "An Exposition of the New Testament"
**Copyright**: Public Domain
**Availability**: Multiple public domain archives

### Characteristics
- **Style**: Scholarly and detailed
- **Approach**: Technical and exegetical
- **Length**: Extensive (often 1000-3000 words per verse)
- **Focus**: Original language analysis, historical context
- **Theological Perspective**: Reformed Baptist

### Content Structure
- Greek word analysis
- Historical and cultural context
- Theological interpretation
- Cross-references
- Doctrinal explanations

### Data Source URLs
- Bible Study Tools: `https://biblestudytools.com/commentaries/gills-exposition-of-the-bible/`
- StudyLight.org: `https://studylight.org/commentaries/geb/`
- Archive.org: Digitized historical editions

## Processing Requirements

### Text Cleaning
- Remove HTML formatting and tags
- Standardize quotation marks and dashes
- Handle footnotes and references
- Preserve paragraph structure

### Verse Linking
- Map commentary to specific verses
- Handle commentary that spans multiple verses
- Create verse-specific extracts
- Maintain source attribution

### Content Segmentation
- Break long commentary into logical sections
- Identify theological themes and tag them
- Create preview text (first 200 characters)
- Maintain expandable full text

### Quality Assurance
- Verify verse mapping accuracy
- Check for complete NT coverage
- Validate theological tag consistency
- Ensure proper source attribution

## Theological Tagging System

### Common Tags
- **Historical**: cultural_context, historical_background, jewish_context
- **Theological**: trinity, salvation, eschatology, covenant, prophecy
- **Literary**: genealogy, parable, miracle, discourse, narrative
- **Doctrinal**: justification, sanctification, predestination, grace
- **Practical**: ethics, discipleship, prayer, worship, fellowship

### Usage
- Each commentary section tagged with 1-5 relevant tags
- Tags enable thematic connections across passages
- Support for future search and filtering features
- Consistent vocabulary across both sources

This dual-commentary approach provides both devotional insight (Henry) and scholarly depth (Gill), giving users comprehensive understanding of selected passages.
